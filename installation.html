Installation

[NOTE to MacOS Users] There is a bundled MacOS package for this which is one of the download options. This bundled package may be installed by dragging the application where you want to install it on your Mac, just like any MacOS application. However, for this kind of software you should probably know more about what the software is doing. You are encouraged to do the installation more generically as described below. This will also make sure you are running from a copy where you can read the source code for what is running.

  1. Install python and graphics libraries:

    Linux

    Required packages (install with system package manager): dpkg-dev, build-essential, python3.0-dev (or newer), libjpeg-dev, libtiff-dev, libsdl1.2-dev, libgstreamer-plugins-base0.10-dev, libnotify-dev, freeglut3, freeglut3-dev, libsm-dev, libgtk-3-dev, libwebkitgtk-3.0-dev

    If available the following newer packages might work a bit better: libgstreamer-plugins-base1.0-dev, libwebkit2gtk-4.0-dev, libsdl2-dev

    MacOS

    Install a version of Python 3+ from www.python.org.

    Windows

    Install a version of Python 3+ from www.python.org.

  2. Install wxPython from PyPi using the command line in a terminal (eg. pip3 install wxpython ). Additional information is available in the wxPython documentation (see wxpython.org).
  3. Download the .zip of this software and decompress the folder (directory) somewhere convenient.
